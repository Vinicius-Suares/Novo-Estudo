We have many types of formatting, but they have a meaning? Thats why we have tags with and without semantic. Lets see then

Bold / Important

We can bold some important thing just using the tag <b>(without semantic)

We can also bold something showing that its important using the tag <strong>(with semantic)

Italic / Enphasis

We can put some texts in italic using the tag <i>(without semantic)

Also, we can put it with semantic using the tag <em>
